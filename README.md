# Nebula Stream - 雲端音樂與影片播放器 (Cloud Music & Video Player)

一個極致現代感、功能完整的網頁版音訊與影片播放器。採用神秘深紫星空主題與磨砂玻璃擬物化 (Glassmorphism) 設計，內建 HTML5 Canvas 即時音訊頻譜視覺化與滾動歌詞同步功能，並提供本地雲端模擬上傳與歌單分享。

## 🌟 功能特點

1. **極致視覺美學**：
   - 採用磨砂玻璃質感 (`backdrop-filter`)、立體發光特效與平滑過渡動畫。
   - 動態背景光暈隨時間慢速飄移，專輯封面隨音樂播放流暢旋轉。
2. **音訊頻譜視覺化 (Audio Visualizer)**：
   - 使用 Web Audio API 即時解析音頻，並在 Canvas 上繪製隨低音與節奏律動的圓形頻譜 halo 脈衝。
3. **MP3 / MP4 雙模播放**：
   - 播放 MP3 時顯示專輯封面與頻譜；播放 MP4 影片時自動展開獨立視訊視窗。
   - 支援進度跳轉、音量調節、0.5x ~ 2.0x 倍速播放、隨機與循環播放模式。
4. **即時同步滾動歌詞 (LRC Sync)** 🆕：
   - 支援 LRC 格式歌詞。播放時當前行歌詞會自動高亮、放大並平滑滾動置中。
   - 點擊歌詞的任一行，播放器會自動跳轉至該句歌詞的時間點播放。
   - 內建「編輯歌詞」彈窗，使用者可隨時匯入或修改一般歌詞與帶有時間軸的 LRC 歌詞。
5. **本地雲端模擬與標籤解析**：
   - 支援將檔案拖曳上傳至媒體庫，後端自動解析 ID3 標籤讀取歌名、歌手、專輯以及內嵌專輯封面與歌詞。
6. **歌單管理與分享**：
   - 可建立自訂歌單、將歌曲加入或移出歌單。
   - 支援**一鍵複製分享連結**（例如 `?playlist=id` 或分享整個歌曲庫），其他人開啟連結即可在雲端加載相同的歌單！
7. **媒體刪除管理** 🆕：
   - 歌曲列提供紅色垃圾桶圖標，一鍵從伺服器永久刪除檔案並清理磁碟空間。

## 🛠️ 技術棧

- **前端 (Frontend)**: Vanilla HTML5, CSS3, JavaScript (ES6), Web Audio API, Canvas API, Lucide Icons
- **後端 (Backend)**: Node.js, Express, Multer, `music-metadata` (解析 ID3 標籤)
- **資料庫 (Database)**: 本地輕量化 JSON 資料庫 (模擬雲端)

## 🚀 快速開始

### 1. 安裝依賴
在專案根目錄下執行：
```bash
npm install
```

### 2. 啟動伺服器
運行伺服器：
```bash
node server.js
```

### 3. 開啟網頁
在瀏覽器輸入以下網址即可開始使用：
`http://localhost:3000`
